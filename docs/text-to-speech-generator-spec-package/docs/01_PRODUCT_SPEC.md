# Product Spec

## Product Name

Text to Speech Generator

## Primary User Strategy

Main strategy: SEO-first traffic acquisition.  
Secondary scenario: students and reading-efficiency users.

## Primary Workflow

1. Visitor lands on `/` or a SEO landing page.
2. Visitor pastes text.
3. Visitor selects language and voice.
4. Visitor completes Turnstile if using guest preview.
5. Visitor generates preview audio.
6. Visitor signs in to download MP3.
7. Registered user consumes monthly quota or Extra Credits on generation.
8. User downloads or replays existing generated audio without additional charge.

## v1 UI Requirements

The first screen must be a usable generator.

Required controls:

- Text area
- Character counter
- Language selector
- Voice selector
- Generate Preview button
- Audio player
- Playback speed selector
- Download MP3 button
- Sign-in CTA when download is unavailable
- Remaining quota display for signed-in users
- Recent history section

## Copy Constraints

Allowed positioning:

- Online Text to Speech Generator
- Free preview
- MP3 download after sign in
- English and Spanish official support
- Beta language support for selected languages

Forbidden claims:

- Unlimited free generation
- 100+ languages
- 500+ voices
- Celebrity voices
- Voice cloning
- No limits

## Page-specific Tool Behavior

All SEO pages reuse the same generator component but may set different default sample text and content blocks.
