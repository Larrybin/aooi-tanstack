# File List

- `handlers/`: AI、Email、Payment、User、Ads 等 Web route 的依赖组装目录。
- `ads-runtime.ts`: Ads runtime settings reader。
- `ai-runtime.ts`: AI runtime settings and provider bindings。
- `api-context.ts`: 实现 api context TypeScript 逻辑。
- `auth-runtime.ts`: Auth runtime settings and bindings。
- `billing-runtime.ts`: 实现 billing runtime TypeScript 逻辑。
- `cloudflare-bindings.ts`: 读取并注入 TanStack Cloudflare Worker bindings。
- `email-runtime.ts`: Email runtime settings and service composition。
- `filelist.md`: 记录当前目录下一层文件和子目录的大致作用。
