# File List

- `routes/`: Web 应用路由文件目录。
- `server/`: Web 应用服务端运行时代码目录。
- `styles/`: Web 应用样式资源目录。
- `client.tsx`: 实现 client React 组件或页面。
- `filelist.md`: 记录当前目录下一层文件和子目录的大致作用。
- `router.tsx`: 实现 router React 组件或页面。
- `routeTree.gen.ts`: 实现 routeTree gen TypeScript 逻辑。
- `server.ts`: 实现 server TypeScript 逻辑。
- `vite-env.d.ts`: 提供 vite env d 的 TypeScript 类型声明。
